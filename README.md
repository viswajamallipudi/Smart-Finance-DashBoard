-> Smart Finance Dashboard

This project is a simple and interactive Finance Dashboard built as part of a frontend assignment. The goal was to design a clean user interface and demonstrate how data can be handled on the frontend without relying on any backend services.

-> What this project does

The dashboard gives a quick overview of personal finances. It includes:

- A summary of balance, income, and expenses
- A chart to visualize financial data
- A list of recent transactions
- An option to add new transactions dynamically
- A dark mode toggle for better user experience

-> Built with

- HTML
- CSS
- JavaScript
- Chart.js (for data visualization)

-> Key highlights

- Focused on clean and simple UI design
- Used JavaScript to dynamically update the UI
- Implemented basic interactivity like adding transactions
- Designed to be responsive across different screen sizes
- Included a dark mode feature to enhance usability
